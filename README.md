# Шаги для воспроизведения
1. Открыть [код программы](https://github.com/Tsetserev/Dz_1.1_Java.git) в IDEA
2. Запустить Программу
3. Посмотреть на вывод Программы

*Ожидаемый результат:* Программа должга показать что куриные бедра на порцию будет 1.2
*Фактический результат:* Программа показывает что на порцию идет 1 куриное бедро
